# Adhil Shanif — Portfolio

Personal portfolio site showcasing my work as a Full-Stack AI Engineer: LLM applications, autonomous agents, full-stack SaaS, and automation pipelines.

**Live:** [adhilshanif.com](https://adhilshanif.com) *(update with your actual domain)*

## Tech Stack

- **Framework:** Next.js (React 19)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) to view the site locally.

## Project Structure

```
src/
├── app/            # Next.js app router (layout, page, globals.css)
├── components/     # UI components (Hero, About, Projects, Contact, etc.)
└── data/           # Project data and links
```
